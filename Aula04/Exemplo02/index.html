Barra Lateral

CSS é uma linguagem que descreve como os elementos HTML devem ser exibidos no seu navegador.

Veja que o CSS permite controlar fontes, cores, margens, linhas, alturas, larguras, imagens de fundo, posicionamentos, etc. Atualmente a linguagem está na versão 3
